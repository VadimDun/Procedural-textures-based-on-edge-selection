# Procedural-textures-based-on-edge-selection
Репозиторий для ВКР 4 курса.

## Что это

Программа анализирует структуру исходного изображения (границы) и создаёт новую текстуру большего размера, сохраняя характерные элементы.

## Как работает

1. **Анализ**: детектор границ находит контуры, суперпиксели группируют их
2. **Классификация**: группы делятся на крупные, средние и мелкие
3. **Размещение**: элементы случайно расставляются на пустом поле (сначала крупные, потом детали)
4. **Смешивание**: края сглаживаются для бесшовного результата

## Запуск

```bash
./ProceduralTexture.exe  [изображение] [rotation] [ширина высота]
```

### Примеры

```bash
# Тестовое изображение (геометрические фигуры)
./ProceduralTexture.exe 

# Своя текстура, размер 1024x1024
./ProceduralTexture.exe texture.png 1024 1024

# Своя текстура, без поворота элементов
./ProceduralTexture.exe texture.jpg off
```

## Управление

| Клавиша | Действие |
|---------|----------|
| `R` | Перегенерировать текстуру |
| `N` | Сменить seed (новый случайный узор) |
| `Q` | Выйти |


## Требования

- C++17
- OpenCV 4.x (core, imgproc, highgui, ximgproc)
- Файл модели Structured Edge Detection (указывается в Config.h, который необходимо добавить)
